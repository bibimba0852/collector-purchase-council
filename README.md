# コレクター購入評議会

コレクション趣味の購入判断をサポートするWebアプリです。

この公開版は、GitHub Pages / iPhone Safariでの利用を想定した最小構成です。

データはこの端末・このブラウザのlocalStorageに保存されます。  
GitHub PagesやGitHubへ自動送信されるわけではなく、別端末や別ブラウザにも自動同期されません。

端末変更、Safariの履歴やWebサイトデータ削除、誤操作に備えて、設定画面から定期的にJSONバックアップを書き出してください。  
iPhoneでは「ファイル」アプリなどに保存しておくと安心です。

診断レポートはバックアップではなく、データの状態を確認するためのレポートです。  
不具合相談に使う場合は、商品名やメモなど見せたくない内容が含まれていないか、共有前に確認してください。
